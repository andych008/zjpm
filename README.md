# 浙江PM2.5监测数据

从浙江省环境保护厅抓取各城市 PM2.5 数据，计算 AQI 值并显示。

Sinatra app deployed on Heroku: [zjair.herokuapp.com](http://zjair.herokuapp.com)

## 数据来源

[浙江省环境保护厅](http://app.zjepb.gov.cn:8080/wasdemo/search?channelid=121215)
